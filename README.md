# unischedule

## Установка

### Требуемые зависимости

* node.js >= 0.10

### Установка и сборка

```bash
$ git clone git@github.com:h4/unischedule.git
$ cd unischedule
$ npm install
```

Все необходимые файлы будут собраны в каталоге `src`. Главный файл –
`src\index.html`.

### Обновление

```
$ git pull
$ npm run build
```
